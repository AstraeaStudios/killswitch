# Killswitch

killswitch is a proxy api for [hokkqi/yiff](https://wrwlf.de/yiff) and probably various other random image sites

Supported Sites:

- [x] e621.net
- [x] e926.net
- [x] yiff.rest
- [x] api.floofy.dev
- [ ] shibe.online
- [ ] randomfox.ca
- [ ] sheri.bot

If you don't see a site that's supported yet, but should be, open an Issue!
